# Navrhněte digitální vizitku / osobní web: centrovaný layout s celoplošným gradie

Navrhněte digitální vizitku / osobní web: centrovaný layout s celoplošným gradientním pozadím, profilová fotka, celé jméno, pracovní pozice, krátký slogan, řádky kontaktních údajů (email, telefon, lokalita) s ikonami, ikony sociálních sítí (LinkedIn, GitHub, Twitter), zástupný QR kód a tlačítko "Stáhnout vCard". Přidejte jemný animovaný efekt pozadí. Patička s copyrightem. Použijte elegantní tmavé téma s akcentní barvou.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Tailwind CSS is preconfigured through Vite and loaded from `src/tailwind.css`.

Built with React + Vite. Deployed via Jean AI.
